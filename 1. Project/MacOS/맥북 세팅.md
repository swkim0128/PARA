## ⚙️ 시스템 유틸리티 및 생산성 도구

### 1. Alfred 5

- 설명: Spotlight를 대체하는 강력한 런처로, 워크플로우 자동화에 탁월합니다.
- 설치 방법: 공식 사이트에서 설치 파일을 다운로드하여 설치합니다.
- 다운로드 링크: [Alfred](https://www.alfredapp.com/)

1. gitignore
	- 기능: 특정 언어 또는 환경에 맞는 `.gitignore` 템플릿을 검색하고 복사합니다.
	- 명령어 예시: `gi node` → Node.js용 gitignore 복사
	- 설치 링크: [urschrei/alfred-gitignore](https://github.com/urschrei/alfred-gitignore)
2. homebrew
	- 기능: Alfred에서 직접 `brew install`, `brew search` 등을 실행할 수 있게 합니다.
	- 명령어 예시: `brew install wget`
	- 설치 링크: [aitorvs/homebrew workflow](https://github.com/aitorvs/alfred-workflows/tree/master/homebrew)
3. http status code
	- 기능: HTTP 상태 코드(예: 200, 404, 500 등)의 의미를 빠르게 검색합니다.
	- 명령어 예시: `http 418` → “I'm a teapot” 출력
	- 설치 링크: [fniephaus/alfred-http-status-codes](https://github.com/fniephaus/alfred-http-status-codes)
4. kill process
	- 기능: 실행 중인 앱이나 프로세스를 Alfred에서 검색하고 종료합니다.
	- 명령어 예시: `kill Chrome`
	- 설치 링크: [vitorgalvao/Kill Process](https://github.com/vitorgalvao/alfred-workflows/tree/master/Kill%20Process)
5. naver search
	- 기능: Alfred에서 바로 네이버 검색을 실행합니다.
	- 명령어 예시: `naver 맛집`
	- 설치 방법: 아래 설정으로 **Custom Web Search** 생성  
	  - Keyword: `naver`  
	  - URL: `https://search.naver.com/search.naver?query={query}`
6. Dash Integration
	- 기능: Dash 앱과 연동하여 개발 문서를 빠르게 검색할 수 있습니다.
	- 명령어 예시: `dash python list comprehension`
	- 필요 앱: https://kapeli.com/dash
	- 설치 안내: https://kapeli.com/help#alfred
7. Translate (Google Translate)
	- 기능: Alfred에서 Google Translate를 통해 실시간 번역 수행
	- 명령어 예시: `tr 안녕하세요` → Hello
	- 설치 링크: https://github.com/yi-ge/alfred-google-translate
8. QR Code Generator
	- 기능: 입력한 텍스트나 URL을 QR 코드 이미지로 생성
	- 명령어 예시: `qr https://example.com`
	- 설치 링크: https://github.com/pstadler/alfred-qr-code
9. Search Recent Files (Finder)
	- 기능: 최근 열었던 파일이나 폴더를 Alfred에서 직접 검색
	- 명령어 예시: `recent`
	- 설치 링크: https://github.com/vitorgalvao/alfred-workflows/tree/master/Recent%20Documents
10. Clipboard History Manager
	- 기능: 복사한 텍스트 기록을 Alfred에서 관리하고 빠르게 붙여넣기
	- 비고: Alfred Powerpack 기본 기능 (별도 설치 불필요)
	- 활성화 경로: Preferences > Features > Clipboard
11. YouTube Search & Download
	- 기능: YouTube 검색 후 영상 다운로드 링크 생성
	- 설치 링크: https://github.com/epilande/alfred-youtube-workflow
