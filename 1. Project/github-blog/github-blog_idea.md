## github blog page 작성 idea 정리

옵시디언에서 작성한 내용을 특정 툴에 상관없이 사용하기 위해 github blog 퍼블릭을 사용하여 접근할 수 있도록 함.
인터넷이 허용하는 한 정리한 내용에 대해 접근할 수 있도록 함.
로컬에서 작성한 옵시디언 파일 내용 중에서 남들에게 보여줄 수 있을만한 내용에 대해 보여줄 수 있도록 함.

para 경로 중에서 resource 경로에 대상으로 blog content 지정

contents/blog - resource/blog?
resource 내용 중에서는 readwise 내용을 link한 내용도 있음
readwise link 부분은 resource 경로에 포함하지 않음
해당 경로 부분은 어떻게 처리해야 하는지 고민 필요
