### RDBMS

---

관계형 데이터베이스 시스템

테이블기반(Table based)의 DBMS

- 데이터를 테이블 단위로 관리.
    
    → 하나의 테이블은 여러 개의 컬럼으로 구성.
    
- 중복 데이터를 최소화 시킴.
    
    → 같은 데이터가 여러 컬럼 또는 테이블에 존재 했을 경우, 데이터를 수정 시 문제가 발생할 가능성이 높아짐. - 정규화
    
- 여러 테이블에 분산되어 있는데이터를 검색 시 테이블 간의 관계(join)를 이용하여 필요한 데이터를 검색.
