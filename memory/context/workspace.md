## Vault Structure
```
01.Projects/  - 진행 중인 사이드 프로젝트
02.Areas/     - 일기, 블로그, 프롬프트, 설정, 템플릿
03.Resources/ - 기술 학습 노트 (알고리즘, JS, React, Python 등)
04.Archives/  - 완료/비활성 문서
05.Excalidraw/ - 다이어그램
06.Temp_Images/ - 임시 이미지
```

## Conventions
- 노트 언어: 한국어
- 파일명 규칙: 번호 접두사 (01., 02. ...)
- 링크: Obsidian [[위키-링크]] 구문
- 다이어그램: .excalidraw.md 확장자
- 태그로 추가 분류
- 커밋 형식: `DOC: backup YYYY-MM-DD HH:mm:ss`
