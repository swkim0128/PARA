# 노션 주간 일정 관리 스킬

## 개요
'일지 및 회고' 데이터베이스에서 이번 주 일정 페이지를 찾아 조회하고, 사용자가 입력한 일정 내용을 해당 페이지의 적절한 위치에 추가하는 스킬입니다.

## 사용 시나리오
```
사용자: "오늘 오후 3시에 팀 미팅 추가해줘"
→ 이번 주 일정 페이지 찾기
→ 해당 날짜 섹션에 "팀 미팅" 추가
```
