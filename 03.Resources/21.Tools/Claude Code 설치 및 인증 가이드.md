# **Claude Code 설치 가이드 🚀**

가장 먼저 모든 자동화의 시작인 **Claude Code (CLI)** 설치 방법을 아주 간단하고 명확하게 정리해 드릴게요. 최근 도입된 공식 '네이티브 설치(Native Install)' 방식을 사용하면 Node.js 같은 복잡한 환경 세팅 없이 단 1분 만에 설치가 가능합니다\!

## **1\. 사전 준비 (Prerequisites)**

* **OS 지원**: Windows, macOS, Linux 모두 지원합니다.  
* **계정**: Claude Pro 구독 계정 또는 Anthropic Console 권한이 필요합니다. (API를 통해 작동하기 때문이에요.)  
* **Windows 사용자 주의사항**: 원활한 파일 시스템 제어를 위해 **Git for Windows**가 시스템에 설치되어 있어야 합니다.

## ---

**2\. 운영체제별 설치 명령어**

사용 중이신 운영체제에 맞는 터미널(Windows의 경우 PowerShell)을 열고 아래 명령어를 복사해서 붙여넣기만 하시면 됩니다.

### **🍎 macOS 및 🐧 Linux**

Bash

curl \-fsSL https://claude.ai/install.sh | bash

명령어를 실행하면 자동으로 시스템 환경을 파악하여 설치하고 업데이트까지 백그라운드에서 처리해 줍니다.

### **🪟 Windows**

반드시 **PowerShell**을 열고 아래 명령어를 실행해 주세요. (일반 CMD에서는 작동하지 않을 수 있습니다.)

PowerShell

irm https://claude.ai/install.ps1 | iex

**💡 참고 (NPM 설치 방식):**

만약 글로벌 환경 변수 제어가 엄격한 회사 컴퓨터를 사용 중이시거나 기존 방식이 편하시다면, Node.js(18버전 이상) 환경에서 아래 명령어로도 설치가 가능합니다.

npm install \-g @anthropic-ai/claude-code

## ---

**3\. 설치 확인 및 초기 인증 (Login)**

설치가 끝났다면 터미널을 껐다가 다시 켜주세요. (새로운 경로를 인식하기 위함입니다.)

그 다음, 본인의 개발 프로젝트 폴더로 이동하여 실행해 봅니다.

Bash

\# 1\. 작업할 프로젝트 폴더로 이동  
cd my-awesome-project

\# 2\. Claude Code 실행  
claude

**초기 인증 과정:**

1. claude 명령어를 처음 입력하면 인증을 위해 웹 브라우저 창이 자동으로 열립니다.  
2. 브라우저에서 Anthropic 계정으로 로그인하고 접근 권한을 승인(Authorize)합니다.  
3. 터미널로 돌아오면 "Hello\!"라는 인사와 함께 이제부터 자유롭게 AI에게 코딩 작업을 지시할 수 있게 됩니다.
