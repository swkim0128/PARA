### Karabiner-Elements

- 설명: 키 리매핑 도구
- 명령어:

```

brew install --cask karabiner-elements

```

왼쪽 command / option 키 위치 교체

- Karabiner > Simple Modifications > 대상 장치를 외부 키보드로 선택
    - left_command → left_option
    - left_option → left_command
   

home / end 키 설정
   
- Karabiner > Complex Modifications > Add predefined rule 버튼 >  
  import more rules from the internet > home end 검색 > home and end import
    - "Home to Command Left" enabled
    - "End to Command right" enabled

karabiner 설정 iCloud 동기화 방법

```bash
# iCloud 폴더 준비
mkdir -p ~/Library/Mobile\ Documents/com~apple~CloudDocs/karabiner_config

# 기존 설정 백업 및 이동
mv ~/.config/karabiner ~/Library/Mobile\ Documents/com~apple~CloudDocs/config/karabiner

# 심볼릭 링크 생성
ln -s ~/Library/Mobile\ Documents/com~apple~CloudDocs/config/karabiner ~/.config/karabiner

# 다른 Mac에서도 적용하기
rm -rf ~/.config/karabiner
ln -s ~/Library/Mobile\ Documents/com~apple~CloudDocs/config/karabiner ~/.config/karabiner
```


- 설정 파일 위치: `~/.config/ghostty/config`
- 주요 설정:

```
# 여백
window-padding-x = 10
window-padding-y = 10
window-theme = system

# 배경 투명도
background-opacity = 0.9
background-blur-radius = 15

# 커서
cursor-style = block
cursor-style-blink = true

# 편의 기능
mouse-hide-while-typing = true
copy-on-select = true
clipboard-paste-protection = false
scrollback-limit = 10000
confirm-close-surface = false

# macOS
macos-option-as-alt = true
shell-integration = zsh
```

### Neovim

- 설명: Vim 기반의 텍스트 에디터. NvChad 배포판을 사용하며 lazy.nvim으로 플러그인 관리.
- 버전: v0.11.6 (LuaJIT 2.1)
- 설치 명령어:

```bash
brew install neovim
```

- 설정 위치: `~/.config/nvim/`
- 배포판: **NvChad** (https://nvchad.com/)
- 설정 구조:
  - `init.lua` — 진입점, lazy.nvim + NvChad 로드
  - `lua/chadrc.lua` — NvChad 커스터마이징
  - `lua/options.lua` — 기본 옵션
  - `lua/mappings.lua` — 키 매핑
  - `lua/plugins/` — 추가 플러그인
  - `lua/configs/` — 플러그인 설정

- NvChad 초기 설치:

```bash
git clone https://github.com/NvChad/starter ~/.config/nvim && nvim
```

### Tmux

- 설명: 터미널 멀티플렉서. 세션/윈도우/패널 분할, 백그라운드 세션 유지 등 필수 도구.
- 버전: 3.6a
- 설치 명령어:

```bash
brew install tmux
```

- 설정 파일 위치: `~/.tmux.conf`
- 플러그인 매니저: **TPM** (https://github.com/tmux-plugins/tpm)

```bash
# TPM 설치
git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm
```

- 사용 플러그인:
  - `catppuccin/tmux` — mocha 테마
  - `tmux-plugins/tmux-cpu` — CPU 사용률 표시
  - `tmux-plugins/tmux-battery` — 배터리 표시
  - `tmux-plugins/tmux-resurrect` — 세션 저장/복원
  - `tmux-plugins/tmux-continuum` — 자동 세션 저장

- 주요 설정 요약:
  - Prefix: `Ctrl+Space` (Ctrl+b도 유지)
  - 윈도우/패널 번호: 1부터 시작
  - vi 복사 모드, `y`로 pbcopy 연동
  - Pane 이동: `Option+hjkl` (prefix 없이)
  - `Prefix+Tab`: yazi 파일 탐색기 50% 분할 오픈
  - 플러그인 설치: `Prefix + I`

### Obsidian

obsidian 설정 iCloud 동기화
   
- Obsidian은 각 **Vault(폴더)** 안에 .obsidian 디렉토리를 생성하여 설정을 저장함
- 이 폴더 안에는 테마, 플러그인, 단축키, 설정 등이 포함됨
- Vault 자체를 iCloud Drive에 두면 자동으로 설정까지 동기화됨

```bash
mkdir ~/Library/Mobile\ Documents/com~apple~CloudDocs/config/obsidian
mv ~/Documents/{Vault}/.obsidian ~/Library/Mobile\ Documents/com~apple~CloudDocs/config/obsidian

rm -rf ~/Documents/{Vault}/.obsidian
ln -s ~/Library/Mobile\ Documents/com~apple~CloudDocs/config/obsidian/.obsidian ~/Documents/{Vault}/.obsidian
```
